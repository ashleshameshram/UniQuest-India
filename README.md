# 🎓 UniQuest India — Statewise University Finde
A web app that uses University API to fetch and display universities across all states of India

## 📌 About The Project
UniQuest India is a mini web project that allows users to search and explore universities statewise across India using a public REST API.
Instead of manually searching, users can simply select or type a state name and instantly get a list of universities available in that region — all fetched in real-time using an API call.

## 🔌 API Used
This project uses the Hipolabs University API - GET http://universities.hipolabs.com/search?country=india
